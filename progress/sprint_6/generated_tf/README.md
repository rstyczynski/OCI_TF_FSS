# Sprint 6 Generated Terraform

This directory contains Terraform configurations generated by Sprint 6 integration tests.

## Test Directories

- `it1_mount_fss/` - IT-1: Mount FSS export test
- `it2_admin_ops/` - IT-2: Administrator operations test

## Usage

These configurations are generated by the test harness and preserved for operator review after test runs. They show the exact Terraform root modules used by Sprint 6.

The generated roots expose the Sprint 5 stack mount-ready outputs:

- `mount_target_mount_addresses`
- `nfs_mount_sources`

`nfs_mount_sources` is the value used by the Sprint 6 SSH workflow in the form `<mount-address>:<export-path>`.
